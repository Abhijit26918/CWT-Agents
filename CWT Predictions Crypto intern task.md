**Intern position assessment**

The document gives a brief introduction to an assignment to check your qualifications for the internship. Please complete it on your own time and submit it for up to 1 week.

**Project Overview**  
Create a backend Python project using Hermes agents framework that provides search and research tool for creating Ads

**Technical Requirements**  
\- Language: Python  
\- Framework: Hermes Agent \- [https://github.com/nousresearch/hermes-agent](https://github.com/nousresearch/hermes-agent)   
\- LLM Provider: [openrouter](http://openrouter.ai) \+ any free model you want  
\- [apify](https://apify.com/)\- for data scraping (its free)

**Project Scope**  
Develop agents with flow that:

1. Agent that search finds crypto assets next 5min prediction both on polymarket and in kalishi  
   1. Ethereum  
   2. Bitcoin   
2. Agent that using apify fetches last known data for crypto asset (last 1000 bars for example)  
3. Agent that predicts next up/down move using [https://github.com/shiyu-coder/Kronos](https://github.com/shiyu-coder/Kronos)  
4. Risk management agent using Kelly formula  
5. Hermes agnes loop feedback based on the results of the prediction  
6. Think outside the box how you can scale this (for example)  
   1. predicting 1min n+5 and using it on 5min n+1  
   2. internal arbitrage between 15 min up/down vs next 3 5min up/down  
   3. user visibility
